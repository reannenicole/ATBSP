# Operator  Operation                          Example          Evaluates to...
**        Exponent                             2 ** 3           8
%         Modulus/remainder                    22 % 8           6
//        Integer division/floored quotient    22 // 8          2
/         Division                             22 / 8           2.75
*         Multiplication                       3 * 5            15
-         Subtraction                          5 - 2            3
+         Addition                             2 + 2            4

Python follows the standard PEDMAS order of operations.
